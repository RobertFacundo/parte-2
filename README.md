# 🚀 Python Course Landing Page

## 📄 Description
This project is a fully responsive landing page designed for a Python course. The landing page features a hero section, a clear value proposition, and smooth animations. The goal was to create an engaging, user-friendly experience that encourages visitors to enroll in the course.

---

## 🛠 Tech Stack
- ⚛️ React (Vite)
- ⌨️ TypeScript
- 🎨 Tailwind CSS
- 🎬 Framer Motion
- 🧩 Component-based architecture
- 📱 Fully responsive design

---

## 🧠 Architecture & Approach
The project is built with a modular and reusable component-based architecture:
- Layout-level composition for different columns.
- Reusable components like buttons and animated sections for smooth user experience.
- Responsive design that adjusts perfectly for all screen sizes, from mobile to desktop.

---

## 🤖 AI Implementation
AI (ChatGPT) was used as an accelerator throughout the development process:
- Helped generate the background image for the landing page based on a visual description.
- Assisted in syntax writing, improving code structure and consistency.
- Suggested animation patterns using Framer Motion to enhance user experience.
- Supported in responsive design and layout adjustments.

_The development process remained manual, with AI acting as a productivity tool._
